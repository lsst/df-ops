Overview
========
.. Include short summary of application, service, or database

Qserv is an open source, massively parallel, distributed SQL database designed to host the petascale astronomical catalogs of the Rubin Observatory.  QServ is integrated with the US DAC to serve astronomy catalogs in a hybrid deployment and with the USDF.
